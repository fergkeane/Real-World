Modal

A single, tokenized dialog primitive that standardizes every popup on the platform — assembled from five composable slots, accessible by default, and re-themeable through CSS variables. Refactored from the original AddPortModal.

06 — Accessibility

Built-in a11y

The component ships WCAG-aligned behavior so consumers don't re-implement it per modal.

Roles & naming. The overlay carries role="dialog" and aria-modal="true", with aria-labelledby bound to the title (or aria-label when titleless).
Focus trap. Tab / Shift+Tab cycle within the dialog; a focusin guard pulls focus back if it ever escapes.
Return focus. The element that opened the modal is restored on close, so keyboard users never lose their place.
Dismissal. Esc closes (when enabled); background scroll is locked and nested modals are reference-counted.
Visible focus. All interactive elements expose a :focus-visible ring — never outline:none without a replacement.
Reduced motion. Rise / fade animations collapse under prefers-reduced-motion.
07 — Usage

Assembling a modal

Link the two files, then call the high-level API or compose the slots by hand.

Install

<link rel="stylesheet" href="components/rw-modal.css">
<script src="components/rw-modal.js"></script>

High-level — open() with slots

const modal = RWModal.open({
  size: 'md',
  title: 'Add a new Port',
  subtitle: 'Define location and metadata',
  closeOnOverlayClick: false,        // form has unsaved input
  initialFocus: '#field-name',
  body: (el) => {                    // body slot — populate imperatively
    el.appendChild(buildPortForm());
  },
  footer: [                          // footer slot — button variants
    { label: 'Cancel', variant: 'secondary', closeOnClick: true },
    { label: 'Save',   variant: 'primary', onClick: (m) => {
        const name = m.query('#field-name').value;
        if (!name) return false;     // keep open on validation failure
        save(name);
        m.close();
    }},
  ],
  onClose: () => console.log('dismissed'),
});

Low-level — compose the sub-components

const P = RWModal.parts;
const overlay   = P.Overlay({ labelledby: 'm-title' });
const container = P.Container({ size: 'lg' });

container.appendChild(P.Header({ title: 'Settings', titleId: 'm-title' }));
container.appendChild(P.Body(myContentNode));
container.appendChild(P.Footer([
  { label: 'Done', variant: 'primary', onClick: (m) => m.close() },
]));
overlay.appendChild(container);

// mount() wires focus-trap, ESC, scroll-lock and return-focus
RWModal.mount(overlay, { closeOnOverlayClick: true, closeOnEsc: true });

Promise-based confirm

const ok = await RWModal.confirm({
  title: 'Delete portfolio?',
  message: 'This cannot be undone.',
  confirmLabel: 'Delete', danger: true,
});
if (ok) deletePortfolio();
